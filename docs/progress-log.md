# Progress Log

> 学习台账——不按周，按**实际发生**记。一行一条，最新在上。
> 作用：把「学到的东西」和「工作里的用处」双向锚定，跳着学也不脱节、不遗忘。
> 地图与状态在 [`learning-roadmap.md`](./learning-roadmap.md)；细节笔记在 `notes/`。

## 怎么记

每条一行，四段式：

```
日期 | 触发（哪个工作任务/问题） | 点亮节点（地图里的哪个） | 沉淀（notes 路径 or 一句结论）
```

- **触发**：为什么学它——最好是具体工作场景，没有就写"主动预习"
- **点亮节点**：对应地图里的条目，学完记得回地图把 `[ ]` 改成 `[~]`/`[x]`
- **沉淀**：notes 文件路径；小结论直接一句话写这也行
