# Notes Index

用于沉淀每个知识点的学习笔记、实验记录和踩坑复盘。

## 目录（对齐 `docs/learning-roadmap.md` 三分骨架）
- `foundations/`：主干——LLM 运行机制、Context Engineering、同心圆心智模型、决策框架
- `ai-coding/`：用 AI 提效——工具、Spec-Driven、CLAUDE.md、Subagents、评审门禁
- `app-dev/`：把 AI 建进产品——LLM 调用、评测、RAG、Agent、MCP、系统设计

## 命名规范
- 推荐：`YYYY-MM-DD-topic.md`
- 示例：`2026-07-03-chunking-strategy.md`

## 写作建议
- 一次只记录一个知识点
- 必须包含“最小可运行示例”与“验证结果”
- 结尾写“下一步行动”
